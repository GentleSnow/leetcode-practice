## 134. Gas Station


### Solution 1: 两次循环

Runtime : Output Limit Exceeded

```Python
class Solution:
    def canCompleteCircuit(self, gas: List[int], cost: List[int]) -> int:
        for i in range(len(gas)):
            # 判断从i开始能否完成循环
            count = 0
            flag = True
            for j in range(len(gas)):
                # print("i:{}, j:{}, idx:{}".format(i, j, (i+j)%(len(gas))))
                count += gas[(i+j)%(len(gas))]
                count -= cost[(i+j)%(len(gas))]
                if count < 0:
                    flag = False
                    break
            if flag:
                return i
        # 不能完成
        return -1
```


### Solution 2: O(n)

Runtime: 60ms

```Python
class Solution:
    def canCompleteCircuit(self, gas: List[int], cost: List[int]) -> int:
        if sum(gas) < sum(cost):
            return -1
        ans = 0
        count = 0
        for i in range(len(gas)):
            count += gas[i] - cost[i]
            if count < 0:
                count = 0
                ans = i + 1
        return ans
```
