## 227. Basic Calculator II

### Solution 1: 两个栈


Runtime: 120ms

```Python
class Solution:
    def calculate(self, s: str) -> int:
        s = s.replace(' ', '')
        _s = re.split(r'[+-/*]', s)
        nums = [int(i) for i in _s]
        cals = [i for i in s if not i.isdigit()]
        i = 0
        while i < len(cals):
            if cals[i] == '-':
                nums[i+1] = -nums[i+1]
            elif cals[i] == '*':
                nums[i] *= nums[i+1]
                del nums[i+1]
                del cals[i]
                i -= 1
            elif cals[i] == '/':
                nums[i] = int(nums[i]/nums[i+1])
                del nums[i+1]
                del cals[i]
                i -= 1
            i += 1
        return sum(nums)
```
