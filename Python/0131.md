## 131. Palindrome Partitioning


### Solution 1: dfs

Runtime: 88ms


```Python
class Solution:
    def dfs(self, s: str, path: List[str], res: List[List[str]]) -> None:
        if not s:
            res.append(path[:])
            return
        for i in range(1, len(s)+1):
            if s[:i] == s[i-1::-1]:
                path.append(s[:i])
                self.dfs(s[i:], path, res)
                path.pop()

    def partition(self, s: str) -> List[List[str]]:       
        res = []
        self.dfs(s, [], res)
        return res
```
