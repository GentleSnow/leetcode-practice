## 167. Two Sum II - Input array is sorted

### Solution 1: HashMap

Runtime: 72ms

```Python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        hmp = {}
        for i, e in enumerate(numbers):
            el = target - e
            if el not in hmp:
                hmp[e] = i + 1
            else:
                return [hmp[el], i+1]
```


### Solution 2: 双向搜索

Runtime: 80ms


```Python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        lIdx = 0
        rIdx = len(numbers) - 1
        while lIdx < rIdx:
            if numbers[lIdx] + numbers[rIdx] == target:
                return [lIdx+1, rIdx+1]
            elif numbers[lIdx] + numbers[rIdx] < target:       
                lIdx += 1
            else:
                rIdx -= 1
```
