## 206. Reverse Linked List


### Solution 1: 迭代

常规方法。
定义两个指针，一个指向当前结点，一个指向前一个节点。
对于对于每一个节点，将辅助节点等于下一个节点，
当前结点指向前一个节点，
指向前一个节点的指针指向当前节点，
将当前节点等于辅助节点，即下一个节点。

Runtime: 36ms

```Python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        prev = None
        curr = head
        while curr != None:
            nextTemp = curr.next
            curr.next = prev
            prev = curr
            curr = nextTemp
        return prev
```



### Solution 2: 递归

设置递归终止条件head==None or head.next==None，


Runtime: 40ms

```Python
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        if head == None or head.next == None:
            return head
        p = self.reverseList(head.next)
        head.next.next = head
        head.next = None
        return p
```
