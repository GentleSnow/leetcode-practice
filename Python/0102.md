## 102. Binary Tree Level Order Traversal


### Solution 1: 层次遍历

Python的层次遍历。

Runtime: 40ms

```Python
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def levelOrder(self, root: TreeNode) -> List[List[int]]:
        if not root:
            return []
        ans = []
        level = [root]
        while root and stack:
            ans.append([n.val for n in level])
            level = [kid for n in level for kid in (n.left, n.right) if kid]
        return ans
```
