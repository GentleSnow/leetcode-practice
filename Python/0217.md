## 217. Contains Duplicate


### Solution 1: Set

Runtime: 44ms

```Python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        if len(nums) == 0:
            return False
        numsSet = set(nums)
        return len(numsSet) != len(nums)
```


### Solution 2: Sort

Runtime: 64ms

```Python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        nums.sort()
        for i in range(1, len(nums)):
            if nums[i] == nums[i - 1]:
                return True
        return False
```
